# 🤖 Bluetooth Controlled Robot using PIC16F877A

## 📌 Overview

This project presents a Bluetooth-controlled robotic vehicle designed using the PIC16F877A microcontroller. The robot is controlled wirelessly through an Android smartphone using the HC-05 Bluetooth module.

The system receives commands from the mobile application via UART communication and controls the motors using the L298N motor driver.

## ⚙️ Communication

The HC-05 Bluetooth module communicates with the PIC16F877A using UART serial communication.

The robot receives characters from the smartphone application:

| Command | Action |
|---------|---------|
| F | Forward |
| B | Backward |
| L | Left |
| R | Right |
| S | Stop |
